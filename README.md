## Клиентская часть программы обучения ##

**Установка**

Перед установкой должен быть установлен nodejs и npm.
После этого выполнить.
```bash
yarn
```
Будут загружены необходимые пакеты.

**Тестирование**

```bash
yarn test
```

**Запуск**

```bash
yarn start
```
